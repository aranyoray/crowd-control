# CrowdLeaf Web Simulator

Web-based interactive simulator for the CrowdLeaf biomimetic crowd control algorithm.

## Features

- **Interactive Visualization**: Real-time simulation rendering
- **Side-by-Side Comparison**: Watch WITH and WITHOUT CrowdLeaf simultaneously
- **Color-Coded Agents**: Green (CrowdLeaf) vs Red (Standard)
- **Real-Time Metrics**: Track injuries, deaths, density, evacuation progress
- **5 Major Airports**: DFW, ATL, DXB, DEL, IAD

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy to Vercel

```bash
vercel
```

2. Deploy automatically

## Technology Stack

- **Framework**: Next.js 15 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Visualization**: HTML5 Canvas
